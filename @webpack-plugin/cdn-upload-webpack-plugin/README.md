# Cdn Upload Webpack Plugin

Corss platform uploader for webpack build.

## INSTALL

```bash
# use npm
$ npm install --dev @dumlj/cdn-upload-webpack-plugin
# use yarn
$ yarn add --dev @dumlj/cdn-upload-webpack-plugin
# use pnpm
$ pnpm add @dumlj/cdn-upload-webpack-plugin -D
```
