# Seed Cli

Basic cli, the registrar for all dumlj cli.

## INSTALL

```bash
# use npm
$ npm install --dev @dumlj/seed-cli
# use yarn
$ yarn add --dev @dumlj/seed-cli
# use pnpm
$ pnpm add @dumlj/seed-cli -D
```
